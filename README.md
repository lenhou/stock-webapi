# 🇹🇼 台灣股市趨勢分析 Web App

這是一個結合 **FastAPI** 後端與 **前端視覺化介面** 的輕量級台股分析工具。透過 `yfinance` 抓取即時行情，並利用 **20 日均線 (MA20)** 演算法提供即時的技術面趨勢判斷。

## ✨ 核心功能

### 1. 智慧型台股代碼識別
* **自動市場判定**：支援輸入純數字代碼（如 `2330` 或 `6547`），系統會自動循環嘗試上市（.TW）與上櫃（.TWO）市場。
* **格式自動校正**：後端會自動處理大小寫與空白，並支援包含字母的代碼查詢（如 ETF 或特別股）。

### 2. 即時行情與詳細數據
* **完整資訊呈現**：包含目前股價、開盤價、**當日最高價**、**當日最低價**、漲跌幅百分比及成交量。
* **視覺化顏色標示**：根據漲跌幅自動標示顏色（紅漲綠跌），符合台灣股市操作習慣。

### 3. 技術趨勢分析 (MA20)
* **自動計算均線**：系統自動提取歷史數據並計算過去 20 個交易日的移動平均線 (MA20)。
* **多空走向判讀**：
    * **🟢 樂觀 (多頭)**：當前股價 > MA20，代表買氣較強。
    * **🔴 悲觀 (空頭)**：當前股價 < MA20，代表賣壓較重。
    * **🟡 盤整中**：股價位於均線附近橫盤。

### 4. 數據紀錄與一鍵匯出
* **歷史走勢表**：顯示最近 10 個交易日的詳細數據（含開盤、最高、最低、收盤、成交量）。
* **格式化複製**：提供專用的文字區域與一鍵複製功能，方便將數據直接粘貼至 Excel 或 Line/Telegram 進行紀錄。

### 5. 響應式使用者體驗
* **手機/電腦適配**：採用自適應網頁設計，在手機上也能擁有良好的操作視野。
* **非同步查詢 (SPA)**：使用 Fetch API 進行數據交換，查詢時網頁無需重新整理。

## 🛠 技術棧
* **Backend**: Python 3.9+, FastAPI, Uvicorn
* **Data**: yfinance, Pandas
* **Frontend**: HTML5, CSS3 (Flexbox), Vanilla JavaScript
